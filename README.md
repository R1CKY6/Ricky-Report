# **Ricky-Report**
*Advanced Report System | FiveM*

# Features
- Custom UI
- Configurable
- Optimized (0.0ms idle)

# Dependencies
- ESX/QBCore
- screenshot-basic

# Installation
1. Drag `Ricky-Report` in your resource folder
2. Execute SQL
3. Setup config
4. Add `ensure Ricky-Report` in your server.cfg
5. Start the server.
